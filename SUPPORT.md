# Support

This repository is the canonical specification and documentation
hub for the PCCX project.

## Where to ask

- **Bug reports / feature requests** — open an issue in the relevant
  repository (`pccxai/pccx` for spec / docs / cross-cutting issues,
  `pccxai/pccx-v002` for the v002 IP-core package, the board
  integration repo for board-specific bring-up, etc.).
- **Discussion** — see the
  [PCCX Discussions](https://github.com/pccxai/pccx/discussions)
  for design questions and reproducibility help.
- **Security** — see [`SECURITY.md`](SECURITY.md).

## What is not support

This file does not constitute a commercial support contract. The
commercial-track support offering is described in
[`docs/commercial/commercial-track.md`](docs/commercial/commercial-track.md)
as a draft policy; pricing and terms are TBD.
