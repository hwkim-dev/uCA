## Foreign-filing window

| Item | Date |
| --- | --- |
| Korean filing date (priority anchor) | 2026-05-08 |
| Internal Madrid / foreign-filing safety deadline | 2026-11-06 |

The 2026-11-06 deadline is an internal operational target. The
Paris Convention priority window is six months from the original
Korean filing; the project deadline is set against that window with
margin for translation, agent selection, and fee handling.

## Decision items (tasks tracked outside this repo)

- Decide the target jurisdictions among Madrid Protocol members
  (US, EU, JP, CN, UK, SG, others) or a subset.
- Estimate Madrid fees for the chosen subset.
- Decide whether to file directly in non-Madrid jurisdictions if
  needed.
- Decide trademark owner if a PCCX-related entity is formed before
  the foreign filings issue.
- Update [`TRADEMARKS.md`](../../TRADEMARKS.md) after each
  registration decision.

## What is **not** here

This page is public-safe. It does not include:

- Customer numbers (특허고객번호).
- Original filing XML files.
- Payment receipts.
- Applicant contact details.
- Screenshots that would expose personal data.

Those records live under the private docket described in
[`private-disclosure-boundary.md`](private-disclosure-boundary.md).

## Update procedure

Editing this page is reserved for status changes:

1. A KIPRIS update flips an application from pending to registered.
2. A foreign filing is decided / deferred / dropped.
3. A new internal deadline is set.

Each edit is reviewed against the public-safe boundary above before
merging to `main`.
